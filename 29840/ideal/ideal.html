Core Components

CPU (Central Processing Unit)

The CPU is the brain of the computer, responsible for executing instructions and performing calculations.

  • Clock Speed: Measures how many cycles a CPU can execute per second, typically in GHz.
  • Cores: Independent processing units within a CPU. More cores generally allow for better multitasking.
  • Cache: Small, fast memory used to store frequently accessed data.

GPU (Graphics Processing Unit)

The GPU is specialized for rendering images, animations, and video.

  • CUDA Cores/Stream Processors: Parallel processors for handling multiple tasks simultaneously.
  • VRAM: Dedicated memory for storing graphical data.
  • Render Output Units (ROPs): Responsible for writing pixels to memory.

RAM (Random Access Memory)

RAM is the computer's short-term memory, used to store data that is actively being used or processed.

  • Capacity: Amount of data that can be stored, measured in GB.
  • Speed: How quickly data can be read or written, measured in MHz.
  • Latency: The delay before a transfer of data begins following an instruction for its transfer.

Storage (HDD/SSD)

Storage devices are used for long-term data retention.

  • HDD (Hard Disk Drive): Uses magnetic storage to store and retrieve digital information.
  • SSD (Solid State Drive): Uses integrated circuit assemblies as memory to store data persistently.
